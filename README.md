# 项目名称
**Marshoai Language Pack**

---

## 项目内容
对[Nonebot-Marshoai](https://github.com/LiteyukiStudio/nonebot-plugin-marshoai) 的所有可供国际化文本进行多语种翻译.

---

## 采用工具
- CGPT 3.5
- Baidu Translate
- Bing Translate
- 人工润色
